# Notaria Word Add-in Installer

## How to Install

1. Put `manifest.xml` and `Install-NotariaAddin.bat` in the same folder on the computer
2. Double-click `Install-NotariaAddin.bat`
3. Wait for completion
4. Restart Microsoft Word
5. The add-in appears in the HOME tab

## What It Does

- Copies `manifest.xml` to the correct Office folder
- Enables Developer Mode in Office
- Configures necessary registry entries
- Word automatically loads the add-in on restart

That's it. Simple.
